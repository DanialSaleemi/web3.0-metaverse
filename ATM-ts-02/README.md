A basic structure of **banking ATM** which is not yet supported by any API calls or DB Connectivity.
But can be used as a skeleton. It contains following functions:
#### 1. Login
#### 2. Balance Inquiry
#### 3. Cash Withdrawl
#### 4. Cash Deposit
#### 5. Funds Transfer
#### 6. Utility Payments
